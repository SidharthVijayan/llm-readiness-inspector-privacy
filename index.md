Privacy Policy – LLM Readiness Inspector

1. Data Collection

The extension processes the following data:

Website content (HTML, text, metadata) of the active tab

This data is accessed only when the user actively runs the extension.

2. Data Usage

The collected data is used solely to:

Analyze webpage structure and semantic quality
Generate an LLM readiness score
Provide optimization suggestions
3. Data Storage
No user data is stored permanently
No browsing history is tracked
No personal data is collected
4. Data Sharing
No data is sold, shared, or transferred to third parties
No data is used for advertising or profiling
5. Remote Processing
All analysis is performed locally in the user’s browser
No website content is transmitted to external servers

6. Permissions Justification
activeTab: Used to access the current page when user clicks the extension
scripting: Used to run analysis scripts on the current page

8. User Control

The extension only runs when explicitly activated by the user.
