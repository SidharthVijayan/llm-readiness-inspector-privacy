Privacy Policy – LLM Readiness Inspector

Effective Date: April 28, 2026

This extension analyzes webpage content to evaluate readiness for Large Language Models (LLMs).

1. Data Collection

The extension accesses website content (HTML, text, metadata) of the active tab only when the user activates it.

2. Data Usage

Data is used only to generate an LLM readiness score and provide optimization suggestions.

3. Data Storage

No data is stored, logged, or retained.

4. Data Sharing

No data is sold, shared, or transferred to any third parties.

5. Processing

All analysis is performed locally in the user’s browser. No data is sent to external servers.

6. Permissions

  • activeTab: Access current page when user clicks extension
  • scripting: Run analysis scripts on the page

7. User Control

The extension only runs when explicitly activated by the user.
